# CutFlow - 비디오 에디터

웹 기반 비디오 편집 애플리케이션입니다. 타임라인 기반 편집, 텍스트 오버레이, 그래픽 효과 등을 지원합니다.

## 주요 기능

- 🎬 **미디어 파일 관리**: 비디오, 오디오, 이미지 파일 업로드 및 관리 (IndexedDB 사용)
- 📝 **타임라인 편집**: 드래그 앤 드롭으로 클립 배치 및 길이 조정
- ✏️ **텍스트 오버레이**: 다양한 스타일의 텍스트 추가 및 편집
- 🎨 **그래픽 효과**: 반짝임, 줌, 페이드, 블러 효과 적용
- 📤 **비디오 내보내기**: FFmpeg.wasm을 사용한 클라이언트 측 비디오 내보내기

## 기술 스택

- **프론트엔드**: React, Vite, Tailwind CSS
- **백엔드**: Express.js (선택사항)
- **비디오 처리**: FFmpeg.wasm (클라이언트 측)
- **스토리지**: IndexedDB (로컬 파일 저장)

## 설치 및 실행

### 필수 요구사항

- Node.js 18 이상
- npm 또는 pnpm

### 설치

```bash
# 의존성 설치
npm install
# 또는
pnpm install
```

### 개발 서버 실행

**프론트엔드만 실행 (권장 - 클라이언트 측 처리):**
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

**프론트엔드 + 백엔드 동시 실행:**
```bash
npm run dev:all
```

또는 별도 터미널에서:
```bash
# 터미널 1 - 프론트엔드
npm run dev

# 터미널 2 - 백엔드 (선택사항)
npm run dev:server
```

### 빌드

```bash
npm run build
```

## 사용 방법

### 1. 로그인
- 로그인 페이지에서 이메일 입력 후 로그인
- 데모 계정으로 바로 사용 가능

### 2. 파일 업로드
- 왼쪽 패널의 "파일 업로드" 버튼 클릭
- 비디오, 오디오, 이미지 파일 선택
- 파일은 브라우저의 IndexedDB에 저장됩니다

### 3. 타임라인 편집
- 업로드된 파일을 클릭하여 타임라인에 추가
- 클립을 드래그하여 위치 변경
- 클립 양쪽 끝을 드래그하여 길이 조정
- 재생 헤드를 드래그하여 특정 시간으로 이동

### 4. 텍스트 추가
- 오른쪽 패널의 "텍스트 추가" 버튼 클릭
- 텍스트 입력 후 추가
- 화면에서 텍스트를 드래그하여 위치 변경
- 텍스트 클릭하여 스타일 편집 (색상, 크기, 폰트)

### 5. 그래픽 효과 추가
- 오른쪽 패널의 효과 버튼 클릭 (반짝임, 줌, 페이드, 블러)
- 각 효과는 타임라인에 독립적인 클립으로 생성
- 효과 클립을 드래그하여 시간 범위 조정
- 강도 슬라이더로 효과 강도 조절

### 6. 비디오 내보내기
- 하단의 "내보내기" 버튼 클릭
- FFmpeg.wasm을 사용하여 클라이언트 측에서 처리
- 완료 후 다운로드

## 아키텍처

### 하이브리드 방식
- **파일 저장**: IndexedDB (클라이언트 측)
- **비디오 처리**: FFmpeg.wasm (클라이언트 측)
- **서버**: 선택사항 (원격 서버 연동 시 사용)

### 파일 구조
```
src/
├── App.jsx              # 메인 컴포넌트
├── utils/
│   ├── indexedDB.js     # IndexedDB 유틸리티
│   └── ffmpegClient.js  # FFmpeg.wasm 클라이언트
└── ...
```

## 문제 해결

### 파일 업로드가 안 될 때
- 브라우저 콘솔(F12)에서 오류 확인
- IndexedDB 저장 공간 확인
- 브라우저 캐시 클리어

### 비디오 내보내기가 느릴 때
- FFmpeg.wasm은 대용량 파일 처리 시 시간이 걸릴 수 있습니다
- 파일 크기를 줄이거나 짧은 클립으로 나누어 편집하세요

### 효과가 작동하지 않을 때
- 효과 클립이 타임라인에 올바르게 배치되어 있는지 확인
- 재생 헤드가 효과 클립의 시간 범위 내에 있는지 확인
